Currently developing a custom game engine in C++ using opengl.
Aims to recreate some of the aspects of the game "Straftat". 
Straftat is one of my most favourite movement shooters because it includes:
  1. Extremely fast paced movement.
  2. Huge weapon variety.
  3. General hilarity when played with friends.
  4. Serious competitive aspect.
  5. Simplicity

My game aims to recreate all of these features.
